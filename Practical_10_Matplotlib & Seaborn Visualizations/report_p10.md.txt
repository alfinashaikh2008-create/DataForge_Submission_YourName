# Practical 10: Matplotlib & Seaborn Visualizations

## Objective
To generate publication-ready statistical plots using Matplotlib and Seaborn.

## Dataset
Sample Superstore dataset

## Visualizations

### 1. Distribution Plot (Sales)
- Used sns.histplot()
- Shows distribution of sales values
- KDE curve helps understand density pattern

### 2. Boxplot (Profit)
- Used sns.boxplot()
- Identifies outliers in profit values
- Displays median, quartiles, and extreme values

### 3. Correlation Heatmap
- Used sns.heatmap()
- Shows correlation between numerical variables
- Helps identify relationships between features

## Observations
- Sales distribution is right-skewed
- Profit contains outliers (both positive and negative)
- Some variables show moderate correlation

## Conclusion
Seaborn and Matplotlib provide powerful tools for statistical data visualization. These plots help in understanding distribution, detecting outliers, and analyzing relationships between variables.
